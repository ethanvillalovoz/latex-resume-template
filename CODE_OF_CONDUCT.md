# Code of Conduct

## Our Standard

This project is intended to be welcoming, practical, and useful. Contributors are expected to communicate respectfully, assume good intent, and keep discussions focused on improving the template, documentation, or user experience.

Examples of welcome behavior:
- Giving specific, constructive feedback.
- Opening issues with clear context and reproduction steps.
- Respecting different levels of LaTeX, resume, and academic CV experience.
- Keeping pull requests focused and easy to review.

Examples of unacceptable behavior:
- Harassment, insults, threats, or personal attacks.
- Dismissive comments about someone else's background, experience, or career goals.
- Spam, trolling, or intentionally disruptive behavior.
- Publishing private information without consent.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and any community spaces associated with this repository.

## Enforcement

Maintainers may remove comments, close issues, block users, or decline contributions that violate this code of conduct. If you see behavior that makes the project less welcoming, contact the maintainer through the email listed in the README.
